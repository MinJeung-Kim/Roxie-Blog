## 🚩프로젝트 소개

- Next.js와 TypeScript를 사용해서 구현한 Blog겸 포트폴리오.
- 블로그에 구현한 기능에 대한 내용 정리.
- [스터디](https://crossfit.gitbook.io/study/)에서 공부한 내용을 바탕으로 구현한 기능과 트러블슈팅에 대한 내용 정리.
- 사이트 : https://roxie-blog.vercel.app/
